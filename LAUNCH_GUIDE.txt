🔄 SWITCHED FROM DISCRETE CYCLES TO CONTINUOUS OPERATION

  OLD MODEL (v1.0):
    - Cronjob every 120 minutes
    - One cycle at 14:00, sleep until 16:00
    - Inefficient: 2 hours of idle waiting
    - Discrete: No overlap between cycles

  NEW MODEL (v2.0):
    - Runs forever (daemon loop)
    - Adaptive cycles: 2-5 min per cycle + sleep 60-180s
    - Always available: 5-12 min time-to-action
    - Continuous: Next cycle begins immediately after sleep

🚀 HOW TO START

  Option 1: Test run (foreground, Ctrl+C to stop)
    autopilot-continuous-daemon

  Option 2: Background daemon
    nohup autopilot-continuous-daemon > logs/daemon.log 2>&1 &
    (Run from any directory; automatically uses AUTOPILOT_HOME)

  Option 3: Systemd service (permanent)
    Create ~/.config/systemd/user/autopilot.service
    systemctl --user enable autopilot.service
    systemctl --user start autopilot.service
