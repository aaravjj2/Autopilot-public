**Title:** APEX: AI-Governed Cross-Market Arbitrage Terminal (Paper-Only)

**Abstract (≤ 150 words):**

Prediction-market arbitrage is fragmented across Kalshi and Polymarket, while most automation tools optimize for speed—not governance. APEX is an institutional-style **paper-only** terminal that combines real-time cross-venue arb detection, a **multi-agent decision panel** (specialists, judge, adversarial checks), and a **14-check risk gate** where `R01` paper enforcement runs first on every path. Operators see live Arb Radar, streaming AI thesis, and append-only audit logs. APEX FinOps Agent (Gemini + Agent Builder + Arize MCP) runs multi-step missions: scan, explain gates, execute or reject paper trades—with full traceability. Built for FinTech innovation and cybersecurity-style fail-closed controls, not unchecked bots.

**Metrics (demo mode):**

- 8+ seeded cross-market opportunities per session
- 14 risk gates evaluated per paper trade
- 100% paper trading — no live execution paths
- Sub-second arb stream via WebSocket JSON Patch
